\# Facial Stress Detection



This project detects facial stress using computer vision and machine learning.



\## Project Overview

The system uses facial features captured via webcam to classify stress or not stressed.



\## Technologies Used

\- Python

\- OpenCV

\- Machine Learning



\## How to Run

1\. Clone the repository

2\. Install dependencies

3\. Run `webcam.py`
